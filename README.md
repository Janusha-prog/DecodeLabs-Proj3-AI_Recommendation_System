# DecodeLabs-Proj3-AI_Recommendation_System
AI recommendation system using cosine similarity for tech career suggestions.

# AI Recommendation Logic System 🚀
This project is a simple AI-based recommendation system developed using Python and Cosine Similarity. The main aim of this project is to recommend the most suitable tech career role based on the skills entered by the user.

In this project, the user can enter skills such as Python, Machine Learning, SQL, HTML, CSS, AWS, Docker, and more. The system compares the entered skills with predefined career skill sets and recommends the best matching career role.

The project also suggests:
* Recommended course
* Skills to learn next
* Career roadmap
* Motivation message
* 
This helped me understand how recommendation systems work in real-world platforms like Netflix, Amazon, Spotify, and LinkedIn.

# How the Project Works
# 1. User Skill Input
The user enters technical skills through the console.
Example:
python machine learning sql

# 2. Dataset Creation
A small dataset of different technology career roles and required skills is created using Python dictionaries and Pandas DataFrame.

# 3. Text Vectorization
CountVectorizer is used to convert text skills into numerical vectors so the computer can compare them.

# 4. Similarity Matching
Cosine Similarity is used to calculate how similar the user skills are to each career role.

# 5. Recommendation Output
The system displays:
* Best matching career role
* Match score
* Recommended course
* Skills to learn next
* Career roadmap
